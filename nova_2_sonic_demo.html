🚀 Amazon Nova 2 Sonic

Bidirectional streaming speech model for real-time voice AI applications.

⚠️ Key Difference: Nova 2 Sonic requires mediaType in all audio configs and only accepts SPEECH input (not text).

🎯 Quick Start
  1. Enable amazon.nova-2-sonic-v1:0 in AWS Bedrock console
  2. Install Python 3.12+ (required for Smithy SDK)
  3. Install: pip install boto3 aws-sdk-bedrock-runtime
  4. Configure AWS credentials
  5. Run the demo scripts in this directory

📝 TTS System Prompt (How it Works)

Since Nova 2 Sonic only accepts SPEECH input, we use a system prompt override trick:

# System prompt to force TTS
system_prompt = f'''
IMPORTANT: Ignore any audio input.
Instead, speak this EXACT text:

"{text}"

Say only this text, word for word.
Do not add or change anything.
'''
💡 Trigger Audio Required: Nova 2 Sonic needs real speech audio (not silence) to trigger Voice Activity Detection. We send a 2-second audio clip.

# Audio Input Configuration

  'sampleRateHertz': 16000,
  'sampleSizeBits': 16,
  'channelCount': 1,
  'audioType': 'SPEECH',
  'encoding': 'base64'
}
