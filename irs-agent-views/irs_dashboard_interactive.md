# IRS Agent Guide - Interactive View

## View ID
`4b281dc3-bf4f-4e48-b27a-e17ea1fdb954`

## Features
- **Click section header** → Expands detailed content
- **Click again** → Collapses back
- Each section includes:
  - ☐ Authentication checklists
  - 📋 Step-by-step procedures
  - ⏱️ Processing time tables
  - ⚠️ Required data capture reminders
  - 📖 Policy links (IRM references)

## 5 IRS Use Case Sections

### 1. 💰 Refund Status Inquiry (HIGH VOLUME)
- Auth: SSN + Filing Status + Refund Amount + Tax Year
- IDRS: `IMFOL`, check `TC 846` or `TC 840`
- Processing times table
- IRM 21.4.1

### 2. 🛡️ Identity Verification 5071C (SECURITY PRIORITY)
- **Fraud indicators** with escalation instructions
- Mandatory verification items checklist
- KBA documentation requirements
- IRM 25.23.2

### 3. 📄 Payment Plan / Installment Agreement
- Agreement types table (Guaranteed, Streamlined, Non-Streamlined, PPIA)
- Fee structure
- Direct Debit savings ($76)
- IRM 5.14.1

### 4. 📑 Transcript Requests
- Direct to IRS.gov/transcripts first
- Transcript types table
- Processing times
- IRM 21.2.3

### 5. 📬 IRS Notice Explanation (HIGH VOLUME)
- Common notice types table (CP14, CP2000, LT11, etc.)
- **LT11 = URGENT levy notice**
- Response deadline guidance
- IRM 21.3.1
